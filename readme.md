- Pull request template
  - [`.github/PULL_REQUEST_TEMPLATE.md`][pull_request]
- Code of Conduct
  - [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md)
- Contribution guidelines
  - [`CONTRIBUTING.md`](CONTRIBUTING.md)
- Support guidelines
  - [`SUPPORT.md`](SUPPORT.md)
